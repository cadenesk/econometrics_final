# Partisan Lean in Congressional Districts

---

## Overview

This project analyzes **partisan lean across U.S. congressional districts** using Census data and election results. It examines how demographic and socioeconomic factors relate to political alignment.

---

## Methodology

* Merged U.S. Census data with district-level election results
* Measured partisan lean using vote share
* Applied regression analysis to evaluate relationships

---

This project is for academic and research purposes.
